# Wave C1 reading-quality mockups

These bitmap mockups explore small, presentation-only improvements for the Pawn Diary tab. They
preserve the existing accordion, year paging, saved diary entries, and DLC independence.

## First pass

- `01-quiet-hierarchy.png` — separates date, title, body, quote, linked perspective, and provenance
  into a calmer reading hierarchy.
- `02-day-timeline.png` — tests a single day heading and subtle sequence rail for orientation.
- `03-focused-reading.png` — gives one expanded entry a stronger reading state, with compact nearby
  entries below it.

## Wave-grounded pass

- `04-cue-pages-copy.png` — C4 + E4 + F2: cue-specific page washes and header rules, styled date
  lines with cue glyphs, and a player-visible copy action in the existing footer seam.
- `05-quadrum-dividers.png` — C2 + C4 + E4 + F2: compact quadrum/season dividers inside the current
  year scroll, retaining ordinary entry cards and accordion behavior.
- `06-russian-reading-stress-test.png` — A5 + B2 + B1 + C2 + C4 + E4 + F2: Russian-facing UI and
  prose with no English schema leakage, plus the visible Wave C1 card treatments.
- `07-combined-direction.png` — selected synthesis: the first concept's restrained palette and
  reading rhythm, an unnumbered decorative left spine, cue icons integrated into the spine nodes,
  and centered quadrum/season separators.
- `08-combined-minimal.png` — subtraction pass on the combined direction: one expanded entry,
  local hairline guide segments, small cue icons, borderless category text, faint quadrum dividers,
  and icon-only footer actions in the authentic RimWorld frame.
- `09-foundation-month-dividers.png` — narrowed direction based on the preferred timeline-card
  foundation: no entry numbering or cue icons, retaining plain circular nodes and adding only
  Aprimay/Jugust quadrum separators.
- `10-neutral-dividers-single-line-header.png` — applies the preferred quiet palette and spacing to
  the numbered-free timeline foundation, with neutral quadrum separators and compact single-line
  `date — title` headers.
- `11-cue-colors-neutral-dividers.png` — restores per-entry cue coding on nodes, slim accents,
  page washes, rules, and category pills while keeping quadrum separators neutral and headers on a
  single line.
- `12-node-state-footer-actions.png` — removes chevrons, moves expansion state into filled versus
  hollow timeline nodes, and restores player-visible copy and regenerate actions in expanded-card
  footers.
- `13-native-footer-icons.png` — refines those footer actions into tiny, unboxed, low-contrast
  RimWorld-style glyphs while preserving the node-based expansion state.

A5 and B2 are represented through more personal prose, not by adding a new identity panel to the
Diary UI. B1 is represented by the Russian mockup: user-facing output is Russian while internal
schema labels remain absent from the rendered page.

The concepts were generated with the built-in image generation workflow using the supplied current
Diary screenshot as the edit target and visual reference.
